# Minos20
いわゆるテ○リス的なやつです。

##注意 : ソースについて
* 習作です。
* バグを豊富に含みます。特に回転がらみは「オカシイ」です。
* 実装が行き当たりばったりのため、整理されていません。

##注意 : ゲーム内容について
* 常にいわゆる「20G」状態です。
* ハードドロップしない限り密着しません。永遠に考えていられます。

##操作方法
* ゲーム開始 : スペースキー  
* 横移動 : カーソルキー左右  
* ハードドロップ : カーソルキー上  
* 左回転 : Zキー  
* 右回転 : Xキー  
* ホールド : 左Shiftキー  
* ギブアップ : ESCキー
